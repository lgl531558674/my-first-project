## 快速开始

### 安装

### 前置要求

- Python 3.8 或更高版本
- Ubuntu / Windows / macOS 均可

### 安装

cd my-first-project
#如果有依赖 （如以后添加了），取消下面这行的注释
pip install -r requirements.txt
